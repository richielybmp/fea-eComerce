This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Clone project

    $ git clone https://github.com/richielybmp/fea-eCommerce.git
    $ cd fea-eCommerce

## Fetch dependencies:
    
    $ yarn install
  
## Running
    
    $ yarn start
